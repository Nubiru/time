# How to access and download data from the Timing Database

Access the Timing Database through its Open Science Framework (OSF) project page and interactive Shiny app for seamless selection and download.[^1]

## Repository Access

Visit the main OSF project at https://osf.io/vrwjz/ to browse 68+ datasets from interval timing studies (e.g., bisection, reproduction tasks).[^1]
Each dataset includes a standardized .csv file with raw trial data and a readme detailing task parameters, intervals, and contributor info.[^1]
Navigate folders for individual downloads or use the linked Shiny GUI at https://timingdatabase.shinyapps.io/DownloadUI/ for custom compilation.[^1]

## Download Steps

1. Go to the Shiny app interface (https://timingdatabase.shinyapps.io/DownloadUI/) and filter by task (e.g., temporal reproduction), species, or interval range.[^1]
2. Select datasets, preview summaries, and compile into a unified .csv format via the GUI's one-click export.[^1]
3. Download the file directly; no login required for public access, though OSF registration enables contributions.[^2]

## Usage Notes

Data follows a common template across tasks like peak interval and temporal order judgments, ideal for scalar property analysis or model fitting.[^1]
The database is live, maintained by the Timing Research Forum (http://timingforum.org), with ongoing submissions.[^3]
For full paper details, read the Behavior Research Methods article (DOI pending, PDF via academic portals).[^2]
